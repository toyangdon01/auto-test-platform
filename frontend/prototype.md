## 1. 整体布局

### 1.1 布局结构

采用经典的**左侧导航 + 顶部标题栏 + 内容区**三栏布局：

```
┌─────────────────────────────────────────────────────────────────┐
│  Logo  │  页面标题                    │  用户信息  │  设置  │
├────────┼────────────────────────────────────────────────────────┤
│        │                                                         │
│  侧    │                                                         │
│  边    │                      内容区                             │
│  导    │                   (Router View)                         │
│  航    │                                                         │
│  栏    │                                                         │
│        │                                                         │
├────────┴────────────────────────────────────────────────────────┤
│  底部状态栏（可选）                                               │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 布局参数

| 区域 | 宽度/高度 | 说明 |
|------|-----------|------|
| 侧边栏（展开） | 220px | 显示图标 + 文字 |
| 侧边栏（收起） | 64px | 仅显示图标 |
| 顶部栏 | 56px | 固定高度 |
| 内容区内边距 | 24px | 上下左右 |
| 底部状态栏 | 32px | 可选 |

### 1.3 导航菜单

```
├─ 首页
├─ 服务器管理
│   ├─ 服务器列表
│   └─ 服务器分组
├─ 脚本中心
│   ├─ 脚本列表
│   └─ 测试类型
├─ 任务管理
│   ├─ 任务列表
│   └─ 创建任务
├─ 测试结果
│   ├─ 结果列表
│   ├─ 结果对比
│   └─ 趋势分析
├─ 报告中心
│   ├─ 报告列表
│   └─ 报告详情
└─ 系统设置
    ├─ 指标定义
    ├─ 判定规则
    └─ 系统配置
```

#### 组件分解

| 组件 | 类型 | 说明 |
|------|------|------|
| StatCard | 统计卡片 | 显示核心指标数字和趋势 |
| TaskTrendChart | 图表 | 近7天任务数量折线图 |
| ServerStatusPie | 图表 | 服务器状态分布饼图 |
| RecentTaskTable | 表格 | 最近任务列表 |

#### 交互说明

1. 统计卡片点击跳转到对应模块
2. 任务趋势图支持切换时间范围（7天/30天）
3. 最近任务表格支持快速跳转任务详情

#### 布局结构（弹窗表单）

```
┌─────────────────────────────────────────────────────────────────┐
│  添加服务器                                              [×]   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 服务器名称 *  [________________________]                 │   │
│  │ 主机地址 *    [________________________] : [22]         │   │
│  │ 所属分组      [选择分组 ▼]                               │   │
│  │ 标签         [+ 添加标签] [test] [x] [prod] [x]         │   │
│  │ 备注         [_________________________________]        │   │
│  │             [_________________________________]        │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  认证信息                                                       │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 认证方式 *    ○ 密码认证  ○ SSH 密钥                    │   │
│  │ 用户名 *      [________________________]                 │   │
│  │ 密码 *        [________________________] 👁              │   │
│  │ 或           [上传私钥文件]                              │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  高级设置                                                       │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 连接超时(秒)  [30]  重试次数 [3]  重试间隔(秒) [5]       │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                          [取消]  [测试连接]  [保存]            │
└─────────────────────────────────────────────────────────────────┘
```

#### 表单字段

| 字段 | 必填 | 类型 | 验证规则 |
|------|------|------|----------|
| 服务器名称 | 是 | 文本 | 1-100字符 |
| 主机地址 | 是 | 文本 | IP或域名格式 |
| 端口 | 是 | 数字 | 1-65535，默认22 |
| 认证方式 | 是 | 单选 | password/ssh_key |
| 用户名 | 是 | 文本 | 1-50字符 |
| 密码/私钥 | 是 | 文本/文件 | 根据认证方式 |
| 所属分组 | 否 | 下拉选择 | - |
| 标签 | 否 | 标签输入 | - |
| 备注 | 否 | 多行文本 | 最多500字符 |

#### 交互说明

1. 认证方式切换时，动态显示密码或私钥上传
2. 测试连接：验证信息填写完整后可点击
3. 保存后自动调用刷新信息接口获取硬件信息

## 5. 交互规范

### 5.1 通用交互

| 场景 | 交互方式 |
|------|----------|
| 新增操作 | 弹窗表单，保存后刷新列表 |
| 编辑操作 | 弹窗表单或跳转编辑页 |
| 删除操作 | 二次确认弹窗 |
| 批量操作 | 勾选后显示批量操作按钮 |
| 状态变更 | 即时更新，无需刷新 |
| 表单提交 | 按钮显示加载状态，完成后提示成功/失败 |

### 5.2 加载状态

| 场景 | 加载方式 |
|------|----------|
| 页面初始加载 | 页面级 Loading |
| 表格刷新 | 表格骨架屏或加载遮罩 |
| 按钮提交 | 按钮加载状态 |
| 异步操作 | 全局 Toast 提示 |

### 5.3 错误处理

| 错误类型 | 处理方式 |
|----------|----------|
| 网络错误 | Toast 提示 + 重试按钮 |
| 参数校验错误 | 表单字段下方显示错误信息 |
| 业务错误 | Toast 提示具体错误信息 |
| 权限错误 | 提示无权限，返回上一页 |

### 5.4 成功提示

- 操作成功：Toast 提示 "操作成功"
- 创建成功：Toast 提示 + 跳转详情页
- 删除成功：Toast 提示 + 刷新列表
