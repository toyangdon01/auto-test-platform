## 🐛 问题与修复

### 问题 1: tags 字段类型错误

**错误信息**:
```
ERROR: column "tags" is of type jsonb but expression is of type character varying
```

**原因**: MyBatis-Plus 的 batch insert 不自动处理 typeHandler

**修复方案**:
```java
// MetricService.java
public void saveBatch(List<Metric> metrics) {
    // 逐个插入（避免 batch insert 的 typeHandler 问题）
    for (Metric metric : metrics) {
        baseMapper.insert(metric);
    }
}
```

**验证**: ✅ 修复后数据正常保存
