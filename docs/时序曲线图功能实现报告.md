## 📊 功能概述

---

## ✅ 已完成工作

### 1. 后端 API

#### 1.1 时序数据查询 API

**接口**: `GET /api/v1/metrics/tasks/{taskId}/timeseries`

**参数**:
- `taskId` (路径参数，必填): 任务 ID
- `metricType` (查询参数，可选): 指标类型（cpu/memory/disk/network）
- `startTime` (查询参数，可选): 开始时间
- `endTime` (查询参数，可选): 结束时间

**响应示例**:
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "timestamps": ["2026-03-18T15:29:57.857885800"],
    "series": [
      {
        "type": "cpu",
        "name": "usage_rate",
        "unit": "%",
        "data": [0.3]
      },
      {
        "type": "memory",
        "name": "used_mb",
        "unit": "MB",
        "data": [15624.0]
      }
    ],
    "total": 6
  }
}
```

**测试结果**: ✅ 通过
- 返回 6 个指标系列
- 时间戳正确
- 数据格式符合前端要求

## 🔧 技术细节

### ECharts 配置

```javascript
{
  title: { text: '指标采集趋势', left: 'center' },
  tooltip: { trigger: 'axis', axisPointer: { type: 'cross' } },
  legend: { type: 'scroll', bottom: 10 },
  grid: { left: '3%', right: '4%', bottom: '15%', containLabel: true },
  toolbox: {
    feature: {
      saveAsImage: { title: '保存图片' },
      dataZoom: { title: { zoom: '区域缩放', back: '区域还原' } },
      restore: { title: '还原' }
    }
  },
  xAxis: { type: 'category', boundaryGap: false, data: [] },
  yAxis: { type: 'value', scale: true },
  dataZoom: [{ type: 'inside' }, { start: 0, end: 100 }],
  series: [] // 动态添加
}
```

### 数据处理流程

```
1. 组件挂载 → initChart() → 初始化 ECharts
2. 加载数据 → loadData() → 调用 API
3. 更新图表 → updateChart() → 设置 option
4. 更新表格 → updateTable() → 转换为表格数据
```
