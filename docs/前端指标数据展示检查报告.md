## 🔍 数据库验证

### 检查指标数据

```sql
SELECT 
  task_id, 
  server_id, 
  metric_type, 
  COUNT(*) as data_points,
  MIN(timestamp) as first_point,
  MAX(timestamp) as last_point
FROM metrics 
WHERE task_id = 225 
GROUP BY task_id, server_id, metric_type;
```

**结果**:
